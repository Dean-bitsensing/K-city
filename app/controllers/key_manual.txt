K-City Tool Key Manual

- 비디오 리사이즈
- 키보드 입력
    - 방향키 왼쪽 : 정지 후 1 scan 전으로 이동
    - 방향키 오른쪽 : 정지 후 1 scan 후로 이동 
    - 방향키 위쪽 : 정지 후 20 scan 후로 이동
    - 방향키 아래쪽 : 정지 후 20 scan 전으로 이동

    - 스페이스 바 : 현재 scan에서 재생 및 정지

    - ESC : 프로그램 종료

    - F1 : 모든 교차로 보는 모드
    - F2 : esterno 교차로 보는 모드
    - F3 : interno 교차로 보는 모드 

    - d : ATM의 Track 정보를 표출하고 싶지 않을 때 필요한 정보 표출/표출 안함
    - delete + 숫자(d 키 누른 뒤 정보 참조) : 해당 ATM의 Track 정보 표출/표출 안함
